The AMS iAQ-Core sensor allows you to use your (datasheet) sensors with ESPHome.

The iAQ-Core sensor module is used to measure VOC levels and provide CO2 equivalent and TVOC equivalent predictions. The data is available via IÂ²C bus.

To use the sensor, first set up an IÂ²C Bus and connect the sensor to the specified pins.

# Example configuration entry
sensor:
  - platform: iaqcore
    address: 0x5A
    update_interval: 60s
    co2:
        name: "iAQ Core CO2 Sensor"
    tvoc:
        name: "iAQ Core TVOC Sensor"
