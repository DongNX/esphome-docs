The ms8607 sensor platform allows you to use your MS8607 (datasheet, Adafruit) temperature, pressure and humidity sensors with ESPHome. An IÂ²C Bus is required to be set up in your configuration for this sensor to work.

      name: Pressure

IÂ²C AddressesÂ¶

The MS8607 digital sensor has two IÂ²C addresses: one for temperature & pressure (0x76), and the other for humidity readings (0x40). They are attached to the same SCL/SDA pins on the package, so if you need to customize the i2c_id, you need to specify it at the top-level for temperature & humidity, and use the same value inside the humidity configuration block.
